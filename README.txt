// =============== 基本操作 =============== //
移动：↑←↓→
攻击：X
大招：空格（需消耗50怒气）


// ================= 玩家 ================= //
初始生命值：120
初始怒气：50
攻击力：8
移动速度：60

// ================= TIPS ================= //
别看怪物攻击力低，但它们的攻击速度极快O(∩_∩)O~
每击中怪物一次积攒怒气4（攻击力的一半）
大招是一整条路径攻击过去，攻击力11，每击中一个敌人恢复1怒气
游戏共有六关，每关有1~3波怪物，大BOSS是带蓝色效果的怪物
打BOSS时各种怪物会随机出现，直至BOSS被击败
